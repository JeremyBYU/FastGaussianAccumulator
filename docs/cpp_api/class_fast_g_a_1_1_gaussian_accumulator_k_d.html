class GaussianAccumulatorKD : public FastGA::GaussianAccumulator<uint32_t>¶

This implements the Gaussian Accumulator with K-D Tree search.

Public Functions

GaussianAccumulatorKD(const int level = 1, const double max_phi = 180.0, const size_t max_leaf_size = 10)¶

Construct a new GaussianAccumulatorKD object. This implements the GaussianAccumulator using a k-d tree method. Its actually pretty fast and can be used, though I recommend GaussianAccumulatorS2.

Parameters

  • level: The refinement level. Iterations of recursive subdivision.

  • max_phi: Latitude degrees from north pole to south pole to include cells in the mesh. Leave at default which includes all of the mesh.

  • max_leaf_size: The max leaf size when building the k-d tre index. Leave at default, its optimal.

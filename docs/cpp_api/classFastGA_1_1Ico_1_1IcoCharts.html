class IcoCharts¶

This is basically my implementation of unwrapping an icosahedron as described in: Gauge Equivariant Convolutional Networks and the Icosahedral CNN - https://arxiv.org/abs/1902.04615.

Public Functions

IcoCharts(const int level_ = 1, const int padding_ = 1)¶

Construct a new Ico Charts object.

Parameters

  • level_: The refinement level. Iterations of recursive subdivision.

  • padding_: Padding around the image, padding = 1 for 3X3 kernel. Please leave at default level, have not correctly made it general.

void FillImage(std::vector<double> normalized_vertex_count)¶

Will take the normalized counts of vertices of the refined icosahedron at copy them to the unwrapped image.

Parameters

  • normalized_vertex_count: These are the normalized counts of the histogram of the vertices of the icosahedron
